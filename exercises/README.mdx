# Building AI-Powered Apps 🤖

Welcome to the workshop! 👋

In this workshop you will add a smart chat assistant to a real online store. The store is already built with React Router. You will use a library called **TanStack AI** to teach the chat how to:

- 💬 Stream answers from a real language model
- 🛡️ Check user input before sending it to the model
- 🎭 Act like a helpful store employee, not a generic chatbot
- 📍 Know which product the customer is looking at right now
- 🛠️ Look up products from the real store database
- 🧠 Suggest a size from real customer reviews
- 📊 Compare products and show the answer as a nice table
- ✅ Ask the user before doing anything important, like adding an item to the cart

By the end you will have a floating chat panel on every product page that can search the catalog, recommend a size, compare items, and add things to the cart with your OK.

This is real production work, not a toy demo. The chat lives in a real app and talks to a real AI model through TanStack AI.
